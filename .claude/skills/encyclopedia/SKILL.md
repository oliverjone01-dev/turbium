# Encyclopedia (v1.1) - Terminology Consistency

## When to invoke

- Любой текст с терминами архитектуры продукта (палитра/линия/коллекция/комплект/артикул/режим продаж)
- Audit публичных страниц (genglass.ru, маркетплейсы, КП, презентации)
- Подготовка новой коммуникации, где упоминаются бренды/палитры/линии/коллекции/комплекты
- Конфликт между авторским желанием и канонической терминологией

## Core terms (v2.1)

### 7-уровневая архитектура
- **Бренд** (Brand): GENGLASS, VALONTI, GENTERO, **Metal-GM** (НЕ GM-METAL), GLASS-MEMORY
- **Категория** (Categoria): зеркала, столы, хранение, перегородки, мягкая мебель, стеклянные доски, комплектующие
- **Линия** (Linea) - сквозной дизайн-код через категории. **TRUBIS** - первая линия GENGLASS
- **Модель** (Modello): EVELIX, RAUNTEL, KUVINO, TRUBIS Oval, etc.
- **Артикул** (SKU): GGT-XX-C-D-SS (столы), GGM-XX-C-S (зеркала), GGP-XX-... (перегородки)
- **Палитра** (Palette): сквозной финиш - NERO, ORO, BIANCO (Core); CIPRIA (Drop весна 2026); INDUSTRIAL (в разработке)
- **Коллекция** (Collezione): сквозной дизайн-концепт с именным автором - первая ожидается Q3-Q4 2026
- **Комплект** (Ensemble): 2-7 артикулов для зоны интерьера - Прихожая/Кабинет/Зона отдыха/Столовая

### Режимы продаж
- **In-Stock (IS)**: 1-3 дня, со склада Домодедово
- **Made-to-Order (MTO)**: 5-30 дней по категории
- **Bespoke**: 20-90 дней, уникальный проект по чертежам

## Common errors (catch & fix)

| ❌ Неправильно | ✅ Правильно |
|---|---|
| «Коллекция NERO/ORO/BIANCO/CIPRIA» | **Палитра** NERO/ORO/BIANCO/CIPRIA |
| «Коллекция TRUBIS» | **Линия** TRUBIS |
| «Розовая коллекция» | Палитра CIPRIA (Drop весна 2026) |
| GM-METAL | **Metal-GM** (с мая 2026) |
| «Ежемесячный комплект» | Комплект (Ensemble), без частоты |
| «Квартальная коллекция» | Квартальный Drop палитры |
| «SKU» (когда имеется в виду тип дизайна) | **Модель** (если тип) / Артикул (если единица) |
| «Интерьерное решение» (как товар) | Scena - способ подачи комплекта, не товар |
| «Коллекция INDUSTRIAL» | Палитра в разработке INDUSTRIAL |

## Audit workflow

1. **GREP** - найти все упоминания контрольных терминов
2. **CONTEXT CHECK** - каждое в контексте: цвет → палитра, дизайн-код через категории → линия, авторский концепт → коллекция
3. **FIX** - заменить с сохранением падежей и согласования
4. **CROSS-CHECK** - после правки прогнать ещё раз (вторая ошибка часто сидит рядом)

## Bridging phrases (для менеджеров)

Когда **клиент** использует старую терминологию - НЕ корректировать в лоб, а переводить через себя:

| Клиент | Менеджер |
|---|---|
| «Из розовой коллекции» | «Наш весенний релиз - палитра CIPRIA, 7 моделей» |
| «Покажите коллекцию TRUBIS» | «Линия TRUBIS - наш бестселлер, столы и стеллажи» |
| «А какие у вас коллекции?» | «Три базовых палитры - чёрная, золотая, белая; плюс весенняя CIPRIA. Первая авторская коллекция выйдет к концу года» |

## Pricing implications

Терминология влияет на цену:
- Палитра Core (NERO/ORO/BIANCO): референсная цена
- Палитра Drop (CIPRIA): +5–15%
- Коллекционная модель: +15–30%
- Лимитированная серия (50 шт, нумерация): доп. +10–20%
