# images/

Папка для растровых картинок карусели (фото, текстуры). НЕ для диаграмм - данные
делаются CSS/SVG (см. `references/images.md`).

## Конвенция имён
- По слайду: `images/01.jpg`, `images/03.jpg` (номер слайда) или осмысленно: `images/cover.jpg`.
- Слот ссылается относительным путём: `<img src="images/03.jpg" ...>`.
- Форматы: jpg / png / webp.

## Источники
1. Свой файл - просто положить сюда под нужным именем.
2. Сгенерить в Nano Banana 2 (Gemini Image) по брифу слота - сохранить сюда.

## Финал
Для пересылаемого HTML вшивать картинки как base64 data-URI (самодостаточно).
Для итераций относительный путь из этой папки работает на экспорте.

Эта папка может быть пустой - тогда слоты показывают брифы-плейсхолдеры.
