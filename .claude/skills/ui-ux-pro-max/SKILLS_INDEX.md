# Claude Design Kit -- Skills Index

> 24 design intelligence skills for Claude Code. Install once, use everywhere.

## Design Intelligence

| Skill | Invoke | Purpose |
|-------|--------|---------|
| UI/UX Pro Max | `/ui-ux-pro-max` | Full design system generator. 67 styles, 96 palettes, 57 font pairings, 25 charts, 13 stacks. BM25 search engine with reasoning. |
| Frontend Design | `/frontend-design` | Production-grade UI with anti-AI-slop principles. Context gathering, bold aesthetics, comprehensive reference library. |
| Teach Impeccable | `/teach-impeccable` | One-time setup -- gathers design context for your project and saves to `.impeccable.md` for all future sessions. |

## Visual Adjustment

| Skill | Invoke | Purpose |
|-------|--------|---------|
| Bolder | `/bolder` | Amplify safe or boring designs. Increase impact, personality, and memorability while maintaining usability. |
| Quieter | `/quieter` | Tone down aggressive designs. Reduce intensity to refined, sophisticated, easier on the eyes. |
| Colorize | `/colorize` | Add strategic color to monochromatic interfaces. Semantic color, accent strategy, OKLCH palettes. |
| Typeset | `/typeset` | Fix font choices, hierarchy, sizing, weight consistency. Modular scales, pairing, readability. |
| Animate | `/animate` | Add purposeful animations and micro-interactions. Timing, easing, entrance choreography, reduced motion. |
| Arrange | `/arrange` | Improve layout, spacing, visual rhythm. Break monotonous grids, fix hierarchy, create intentional compositions. |
| Overdrive | `/overdrive` | Push past conventional limits. Shaders, virtual scrolling, View Transitions, spring physics, scroll-driven animation. |
