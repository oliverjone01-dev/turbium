# Стратегии ставок Яндекс Директ

## Автоматические стратегии

### Оптимизация конверсий
- **Когда:** Есть данные по конверсиям (≥ 10 конверсий/неделю)
- **Настройки:** Целевая CPA или бюджет недели
- **Ограничения:** Средняя CPC, недельный бюджет
- **Подводные камни:** Недостаточно конверсий → стратегия не обучится
- **Рекомендация:** Основная стратегия для зрелых кампаний

### Оптимизация кликов
- **Когда:** Новые кампании, недостаточно данных для конверсий
- **Настройки:** Средняя/максимальная цена клика, недельный бюджет
- **Переход:** При накоплении 10+ конверсий/неделю → переключить на конверсии
- **Рекомендация:** Стартовая стратегия

### Оптимизация рентабельности (ROI)
- **Когда:** E-commerce с e-commerce данными в Метрике
- **Настройки:** Целевая рентабельность (%), недельный бюджет
- **Требования:** Передача дохода в Метрику, ≥ 20 конверсий/неделю
- **Рекомендация:** Для интернет-магазинов с хорошей аналитикой

### Максимум показов по минимальной цене
- **Когда:** Только для медийных/баннерных кампаний
- **Не использовать:** Для перфоманс-кампаний

### Максимум кликов по минимальной цене
- **Когда:** Трафиковые кампании, сбор аудитории
- **Ограничения:** Недельный бюджет обязателен
- **Рекомендация:** Только если нет конверсионных целей

## Ручное управление

### Ручное управление ставками
- **Когда:** Полный контроль, A/B тесты стратегий, очень малый объём
- **Минусы:** Трудозатратно, нет автооптимизации
- **Рекомендация:** Только для опытных специалистов с маленькими аккаунтами

## Дерево выбора стратегии

```
Есть ≥ 20 конверсий/неделю + доход?
├── Да → Оптимизация рентабельности (ROI)
└── Нет
    Есть ≥ 10 конверсий/неделю?
    ├── Да → Оптимизация конверсий (целевая CPA)
    └── Нет
        Новая кампания?
        ├── Да → Оптимизация кликов → через 2-4 недели пересмотр
        └── Нет → Оптимизация конверсий с оплатой за конверсии
```

## Корректировки ставок

| Тип | Диапазон | Когда применять |
|-----|----------|-----------------|
| Устройства (мобильные) | -50% … +300% | По данным конверсий по устройствам |
| Пол | -100% … +300% | Если целевая аудитория конкретного пола |
| Возраст | -100% … +300% | По данным конверсий по возрастам |
| Гео | -100% … +300% | Разная конверсионность по регионам |
| Время | -100% … +300% | Разная эффективность по часам/дням |
| Платёжеспособность | -100% … +300% | Таргетинг на аудиторию с высоким доходом |

## Рекомендации аудита

- **YD49:** Стратегия должна соответствовать целям и объёму данных
- **YD50:** Бюджет должен быть достаточным (≥ 300 руб/день для автостратегий)
- **YD51:** Ограничения CPC не должны быть слишком жёсткими (иначе стратегия не работает)
- **YD28:** Широкое соответствие ТОЛЬКО с автостратегией
