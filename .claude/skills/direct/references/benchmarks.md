## Показатель качества

| Уровень | Значение | Влияние на CPC |
|---------|----------|----------------|
| Низкий | 1-4 | CPC × 1.5-3.0 |
| Средний | 5-7 | CPC × 1.0 (норма) |
| Высокий | 8-10 | CPC × 0.5-0.8 (скидка) |

**Цель:** средний показатель качества по аккаунту ≥ 5, по брендовым ≥ 8.

## Правила оценки на аудите

- CTR поиска < 3% → FAIL (кроме высококонкурентных ниш)
- CTR поиска < 5% → WARNING
- CTR РСЯ < 0.2% → FAIL
- CPC > 2x от среднего по тематике → WARNING
- CVR < 0.5% (поиск) → FAIL
- Показатель качества < 5 → WARNING, < 3 → FAIL
