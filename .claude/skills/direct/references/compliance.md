# Правила модерации и комплаенс Яндекс Директ

## Общие требования к рекламе

### Закон о рекламе (ФЗ-38)
- Реклама должна быть достоверной и не вводить в заблуждение
- Обязательная маркировка «Реклама» (с сентября 2022)
- Передача данных в ЕРИР (Единый реестр интернет-рекламы) через ОРД
- Токен (erid) обязателен для всех рекламных материалов

### Маркировка рекламы (с 2023)
- Все рекламные креативы должны быть зарегистрированы в ОРД
- Яндекс автоматически передаёт данные, но проверять корректность
- Указание рекламодателя и ИНН обязательно

## Запрещённые тематики

| Категория | Статус | Примечание |
|-----------|--------|------------|
| Наркотики | Полный запрет | — |
| Оружие (нелегальное) | Полный запрет | — |
| Порнография | Полный запрет | — |
| Азартные игры (нелицензионные) | Запрет | Лицензированные — с ограничениями |
| Табак | Полный запрет | Включая электронные сигареты |
| Алкоголь крепкий | Запрет в интернете | Пиво — с ограничениями |

## Требующие лицензий/документов

| Категория | Требования |
|-----------|------------|
| Медицинские услуги | Лицензия + дисклеймер «Имеются противопоказания» |
| Лекарства (рецептурные) | Запрет. Безрецептурные — с дисклеймером |
| Финансовые услуги | Лицензия ЦБ, полные условия |
| Микрозаймы (МФО) | Лицензия ЦБ, указание полной стоимости |
| БАДы | Дисклеймер «Не является лекарством» |
| Страхование | Лицензия + полные условия |
| Образование | Лицензия на образовательную деятельность |
| Строительство | СРО для определённых видов работ |

## Требования к объявлениям

### Текст
- Нет CAPS LOCK (кроме аббревиатур)
- Нет восклицательных знаков в заголовке
- Нет грамматических ошибок
- Нет превосходных степеней без подтверждения («лучший», «№1»)
- Нет гарантий результата в медицине/финансах
- Цена должна быть актуальной
- Нет контактной информации в тексте объявления (телефон, email)

### Посадочные страницы
- Страница должна быть доступна
- Содержание должно соответствовать объявлению
- Нет автоматических скачиваний
- Нет вредоносного ПО
- Политика конфиденциальности при сборе данных
- Юридическая информация (ИНН, ОГРН) для коммерческих сайтов

### Изображения (РСЯ)
- Без мелкого текста (< 20% площади)
- Без шокирующего контента
- Без «до/после» в медицинской рекламе
- Соответствие рекламируемому продукту

## Возрастная маркировка

Обязательно указывать возрастное ограничение:
- 0+ / 6+ / 12+ / 16+ / 18+
- Определяется содержанием рекламируемого продукта
