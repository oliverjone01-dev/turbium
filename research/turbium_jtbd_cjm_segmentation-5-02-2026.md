# 1. ОБЩАЯ КАРТА СЕГМЕНТОВ

## 1.1. Методология

Сегментация построена по методологии **Jobs To Be Done (JTBD)** — мы определяем не «кто клиент», а «какую работу он нанимает TURBIUM выполнить».

Ключевой принцип: **один сегмент = одна главная боль = один Job To Be Done**

## 1.2. Четыре ключевых сегмента

```
                              TURBIUM
                          «Элемент вашего роста»
                                  │
        ┌─────────────┬───────────┼───────────┬─────────────┐
        │             │           │           │             │
   ┌────▼────┐  ┌─────▼─────┐ ┌───▼───┐ ┌─────▼─────┐
   │ ХАОТИК  │  │РАЗОЧАРО-  │ │МАСШТА-│ │ НОВИЧОК   │
   │         │  │  ВАННЫЙ   │ │ БАТОР │ │           │
   └────┬────┘  └─────┬─────┘ └───┬───┘ └─────┬─────┘
        │             │           │           │
   «Навести       «Перестать   «Расти      «Понять,
    порядок»       сливать»   быстрее»    с чего
                                          начать»
```

# 2. СЕГМЕНТ 1: ХАОТИК

## 2.1. Job To Be Done

> **«Помоги мне навести порядок в маркетинге, чтобы я понимал, что работает, а что нет»**

### Формула JTBD

```
КОГДА я [управляю бизнесом с хаотичным маркетингом]
Я ХОЧУ [понять, какие каналы реально приносят клиентов]
ЧТОБЫ [перестать тратить деньги вслепую и получить контроль]
```

## 2.2. Портрет сегмента

### Демография и бизнес

| Параметр | Значение |
|----------|----------|
| **Возраст** | 35-50 лет |
| **Пол** | 70% мужчины, 30% женщины |
| **Бизнес** | Услуги, торговля, производство |
| **Возраст бизнеса** | 2-7 лет |
| **Оборот** | 3-30 млн ₽/год |
| **Команда** | 5-30 человек |
| **Маркетинг-бюджет** | 50-200к ₽/мес |

### Текущая ситуация

- Маркетинг «на коленке»: сам собственник + фрилансеры + иногда агентство
- Нет единой системы аналитики
- Решения принимаются интуитивно
- Каналы не связаны между собой
- CRM либо нет, либо ведётся частично

### Психографика

| Аспект | Описание |
|--------|----------|
| **Главная боль** | Не понимает, откуда приходят клиенты и что реально работает |
| **Страх** | «Уеду в отпуск — всё развалится», «Сливаю деньги непонятно куда» |
| **Желание** | Получить ясную картину и контроль над маркетингом |
| **Барьер** | Нет времени разбираться, кажется сложным |
| **Триггер покупки** | Устал тушить пожары, готов заплатить за ясность |

### Внутренний монолог

> *«У меня вроде есть сайт, какая-то реклама крутится, SMM-щик что-то постит... Но я не понимаю, что из этого реально приносит деньги. Каждый месяц трачу 150-200к на маркетинг, а откуда заявки — хрен знает. Таргетолог говорит одно, SEO-шник другое, я киваю и плачу. Хочу наконец разобраться, но времени вникать нет.»*

> **«Помоги мне перестать сливать деньги на маркетинг, который не работает»**

```
КОГДА я [уже потратил много денег на маркетинг без результата]
Я ХОЧУ [понять, где меня обманывают и как это исправить]
ЧТОБЫ [перестать терять деньги и начать получать реальные продажи]
```

## 3.2. Портрет сегмента

### Демография и бизнес

| Параметр | Значение |
|----------|----------|
| **Возраст** | 40-55 лет |

| **Команда** | 10-50 человек |

- Работал с 2-5 агентствами за последние 3 года
- Пробовал разных специалистов: таргет, SEO, контекст, SMM
- Красивые отчёты, но продажи не растут
- Глубокое недоверие к маркетологам
- Готов платить, но хочет гарантий и прозрачности

### Психографика

| Аспект | Описание |

### Внутренний монолог

Я устал от этого цирка. Все эти маркетологи — одинаковые. Обещают золотые горы, а потом "рынок сложный, конкуренция высокая". Хочу понять сам, что работает, чтобы меня больше не разводили.»*

# 4. СЕГМЕНТ 3: МАСШТАБАТОР

## 4.1. Job To Be Done

> **«Помоги мне расти быстрее без пропорционального роста команды и хаоса»**

### Формула JTBD

```
КОГДА я [хочу масштабировать бизнес, но упираюсь в потолок]
Я ХОЧУ [найти способ расти эффективнее, а не просто "больше"]
ЧТОБЫ [удвоить результат без удвоения головной боли и расходов на команду]
```

## 4.2. Портрет сегмента

### Демография и бизнес

| Параметр | Значение |
|----------|----------|
| **Возраст** | 35-50 лет |
| **Пол** | 80% мужчины, 20% женщины |
| **Бизнес** | Услуги, e-commerce, производство, франшизы |
| **Возраст бизнеса** | 5-15 лет |
| **Оборот** | 30-150 млн ₽/год |
| **Команда** | 20-100 человек |
| **Маркетинг-бюджет** | 500к-2 млн ₽/мес |

### Текущая ситуация

- Маркетинг работает, есть стабильный поток клиентов
- Но рост замедлился, упёрся в потолок
- Для x2 роста нужно x3 людей и x5 проблем
- Интересуется автоматизацией и AI
- Готов инвестировать в системы, которые масштабируются

### Психографика

| Аспект | Описание |
|--------|----------|
| **Главная боль** | Линейный рост: больше результата = больше людей = больше хаоса |
| **Страх** | «Масштабирование убьёт качество», «Не справлюсь с управлением» |
| **Желание** | Найти рычаги роста, которые не требуют пропорционального увеличения ресурсов |
| **Барьер** | Скептицизм к «волшебным таблеткам», нужны проверенные решения |
| **Триггер покупки** | Увидел конкретный кейс автоматизации с цифрами |

### Внутренний монолог

> *«Мы делаем 50 млн в год, хочу 100. Но если я найму ещё 5 человек в маркетинг — это +1.5 млн ФОТ в год, новые проблемы с управлением, обучение, текучка... Я уже не справляюсь с текущей командой. Должен быть способ расти умнее. Все говорят про AI — но это хайп или реально работает? Хочу увидеть конкретные примеры, а не теории.»*

## 4.3. Customer Journey Map (CJM)

### Этап 1: ОСОЗНАНИЕ (Awareness)

| Параметр | Описание |
|----------|----------|
| **Триггер** | Планирование на год: понимает, что текущая модель не масштабируется |
| **Мысли** | «Чтобы вырасти x2, мне нужно x3 людей. Это тупик» |
| **Эмоции** | Фрустрация, поиск решения |
| **Действия** | Изучает темы автоматизации, AI в маркетинге, кейсы роста |

# ПРИЛОЖЕНИЯ

## A. Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| **JTBD** | Jobs To Be Done — методология, фокусирующаяся на «работе», которую клиент хочет выполнить |
| **CJM** | Customer Journey Map — карта пути клиента от осознания проблемы до покупки и далее |
| **LTV** | Lifetime Value — пожизненная ценность клиента |
| **Upsell** | Продажа более дорогого продукта существующему клиенту |
| **Лид-магнит** | Бесплатный ценный материал в обмен на контакт |
