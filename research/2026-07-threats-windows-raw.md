# RAW: Карта угроз и окон 07.2026-07.2028 (агент-разведка, 22.07.2026)

Статус: сырой отчёт до сверки ДАТЫ. Метки [ДАННЫЕ]/[ОЦЕНКА]/[ГИПОТЕЗА] в тексте.

## Мертвецы ходячие
1. Копирайтинг-генераторы общего назначения: Jasper ~$120M (2023) → $35-55M (2024), 4 волны увольнений [ДАННЫЕ]
2. Транскрибация/саммари встреч: съедены нативными Zoom/Teams/Meet [ДАННЫЕ]
3. Тонкие GPT-wrappers: 60-70% с нулевой выручкой, прогноз 90% мёртвых к концу 2026 [ОЦЕНКА]
4. Промпт-паки и GPT Store приложения: каннибализированы каденсом OpenAI [ОЦЕНКА]
5. Вводные курсы «как пользоваться ChatGPT»: рынок обучения AI в РФ растёт (2.1 млрд ₽ 1П2025 → прогноз 5.6 млрд к концу года [ДАННЫЕ habr/bothub]), но спрос смещается к узким измеримым программам

Контрпример выживания: DeepL, EvenUp - проприетарные данные/качество, не обёртка. 85% прибыльных AI-стартапов 2025 контролировали проприетарный датасет [ДАННЫЕ по источнику агента].

## Платформы: куда НЕ идти (уже бесплатная кнопка)
- Yandex AI Studio (07.2026): no-code конструктор агентов, пресеты «маркетинговый аналитик», «аналитик тендеров» - ПРЯМАЯ угроза нише «настрою вам агента»
- GigaChat B2B: от 1000 ₽/мес; Битрикс24 BitrixGPT: 100 запросов/мес бесплатно, YandexGPT+GigaChat встроены
- Google Workspace: Gemini во всех платных планах от $7; amoCRM AI слабый (виджеты) - там пусто
- Опасная зона: генерация контента, базовые чат-боты, саммари, «аналитик по клику»

## Смертность моделей
- Micro-SaaS: 54% на верифицированных Stripe-данных = 0 выручки; 70% зарабатывающих <$500/мес; 18% достигают $1-5K MRR; ~92% не доходят до $1M ARR [ОЦЕНКА]
- Vertical SaaS: удержание x2 против горизонтальных; 73% МСБ недовольны «инструментами для всех» [ДАННЫЕ appscrip]

## Регуляторика РФ (критично)
1. Закон о развитии ИИ принят 08.07.2026: маркировка «социально значимого» AI-контента, мягкий режим [ДАННЫЕ vedomosti/kommersant]
2. 289-ФЗ платформенная экономика с 01.10.2026: рост издержек селлеров МСБ = ОКНО для помощи с адаптацией [ДАННЫЕ]
3. Маркировка рекламы ERID/ОРД: штрафы 100-500К ₽ на ОБЕ стороны [ДАННЫЕ]
4. OpenAI закрыт для РФ напрямую; ЕС расширил запреты на AI-сервисы для РФ (01.2026) - архитектура на одном западном API = единая точка отказа. Мультипровайдерность (YandexGPT+GigaChat) обязательна

## Окна 12-24 мес
1. Комплаенс-инструменты под 3 новых закона сразу (маркировка AI-контента + ERID + 289-ФЗ) - платформы решают только свой периметр. Окно 12-18 мес
2. Вертикальные шаблоны агентов поверх Yandex AI Studio/GigaChat («настройка для стоматологии/риелтора») - премия ACV 40-60% за отраслевую настройку [ДАННЫЕ US-прецедент]. Окно 12-24 мес, риск: платформы сами выкатят пресеты
3. Узкие измеримые учебные продукты («встроить AI-агента в процесс продаж и измерить ROI за 30 дней»), не «AI-грамотность» - рынок растёт, вводные курсы коммодитизируются
4. Данные/бенчмарки как продукт: курированные библиотеки сценариев с измеренной конверсией - не копируется релизом фичи

## Anti-pattern чеклист (2 человека, 0 ₽)
1. Enterprise-цикл продаж 6+ мес
2. Обещание поддержки 24/7
3. Один API-провайдер, тем более западный из РФ
4. Ценовая война с бесплатной кнопкой платформ
5. Тонкая обёртка без проприетарных данных/курации
6. Сползание в кастомные проекты «настроим именно под вас»
7. Широкий вводный AI-курс как основной продукт
8. Игнор маркировки: один штраф 100-500К при кассе 0 = смерть компании
9. Продажа на голом обещании «утроит продажи» без данных
