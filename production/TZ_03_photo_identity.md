# ТЗ 03 · Фото-генерация: люди, предметка, айдентика

Для: Nano Banana Pro / Midjourney v7 / FLUX (фотореализм). Привязка: карусели типов «ИСТОРИЯ» и «Человек в свете», имиджевые посты, обложки статей блога, аватары площадок.

## 1. Приём «Человек в свете» (split-light, референс HUBONKT)

Назначение: карусели-истории («Три агентства за год», «День собственника»), имиджевые посты.

```
Cinematic portrait, vertical 4:5. Business owner (40-45, tired but determined), dark
clothing (black turtleneck), dark environment #0A0F0D. DRAMATIC SPLIT LIGHTING: amber
#FF8C42 rim light from left (problem side), mint #4EEAC0 rim light from right (solution
side), face partly in shadow. Shallow DOF, moody, photoreal, medium format look.
FACE MUST NOT BE RECOGNIZABLE as a real person: 3/4 turn away or partial shadow.
NO stock-photo smile, no office clichés.
```
Вариации сюжета: за ноутбуком ночью (экран = единственный тёплый свет) / у окна цеха / рука с телефоном и пропущенными вызовами / силуэт против мятного grid-экрана.

Правило света = правило драматургии: в кадрах «боль» доминирует amber, в кадрах «решение» доминирует mint. Смена баланса света рассказывает историю без слов.

## 2. Предметка «Объект-метафора» (библиотека)

```
Premium product-style macro photo, vertical 4:5, object on dark reflective surface,
deep black background #0A0F0D, subtle teal #00BFA5 grid barely visible behind.
Single light source with <amber #FF8C42 | mint #4EEAC0> glow. Photoreal, 8K,
lots of negative space (upper 40% reserved for text overlay).
```

| Объект | Метафора | Юниты плана |
|---|---|---|
| Монета на ребре, горящая / целая | Слив бюджета / решение | нед 1, обложки ШОКЕР |
| Дырявое ведро со светящимися каплями | Потеря лидов | нед 1, 13 |
| Песочные часы с amber-песком | Время = деньги, рутина | нед 6, 12 |
| Лабораторная колба (пустая/полная) | Формула Tb, реакция роста | рубрика ФОРМУЛА |
| Стикер «перезвонить» на тёмном мониторе | Забытые заявки | нед 1, 14 |
| Чемодан, из которого сыплются визитки | Менеджер уносит базу | нед 7, 9 |
| Разбитая витрина смартфона с уведомлением | Пропущенный клиент | нед 14 |
| Калькулятор с amber-цифрами | Юнит-экономика | нед 4, 21 |

Запреты: криптосимволы на монетах; брендированные предметы (логотипы техники ретушировать); хаос в кадре (один объект = один смысл).

## 3. Айдентика и аватары

Аватар канала/бота (уже в ДНК): элемент Tb в квадратной рамке, мятное свечение, читается в 40px:
```
Minimal app icon, square: periodic table element tile "Tb", thin mint #4EEAC0 border,
deep black background #0A0F0D, atomic number 65 top-left, soft mint glow. Flat, crisp,
readable at 40px. No extra text.
```
Обложки статей блога: тип B (факт-типографика) из ТЗ 01 в горизонтали 1200x630 (OG-image), одна цифра-хук + Tb-марка.

## 4. Консистентность серий

- Один и тот же «герой-собственник» в серии недели: фиксируй seed/референс-изображение и повторяй описание внешности дословно (возраст, одежда, сложение).
- Поверхность предметки одна на всю рубрику (тёмный камень для ШОКЕР, матовый металл для ФОРМУЛА).
- Цветопроба каждого рендера пипеткой: фон в диапазоне #0A0F0D-#10160F, амбер и минт без сдвига в красный/голубой.

## 5. Правовая гигиена

1. Лица не узнаваемы (152-ФЗ + отсутствие модельных релизов + анти-дипфейк гигиена).
2. Не имитировать реальные бренды, вывески, приложения конкретных банков.
3. Скриншоты-стилизации (чат, заметки) собирать как UI-арт, не как подделку реальной переписки; вымышленные имена.
4. Фото «клиентов» не выдавать за реальных клиентов (правило: вымышленные кейсы запрещены; подпись «модельный кадр» при сомнении).

## 6. Чек-лист качества фото

1. Свет = роль (amber боль / mint решение), баланс не случайный
2. Лицо неузнаваемо, поза естественная, без стоковой улыбки
3. Воздух под текст >=30-40% кадра
4. Один объект = один смысл, фон не спорит
5. Цвета в допуске пипеткой
6. Серия консистентна (герой/поверхность/оптика)
